# Zusteller-App

Eine Webanwendung zur Verwaltung und Planung von Zustelltouren.

## Funktionen

- Adressen verwalten
- Orte gruppieren
- Suche
- Google Maps Navigation
- Adressen bearbeiten
- Erledigt markieren
- Fortschrittsanzeige
- Lokale Datenspeicherung

## Technologien

- HTML
- CSS
- JavaScript
- LocalStorage

## Geplante Funktionen

- Routenoptimierung
- OpenStreetMap Integration
- Android-App
- Import / Export
